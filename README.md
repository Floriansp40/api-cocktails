# Installation

Api-Cocktail utilise Node.js .

```sh
$ npm install
```

# Avant tout
Penser à renomer le fichier .env.exemple en .env et d'y mettre ses propre paramètre

# Pré-requis
Une base de donnée Mysql/MariaDb

# Lancement du serveur
```sh
$ npm run dev
/
$ npm run start
```
